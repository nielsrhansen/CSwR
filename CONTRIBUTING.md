# Contributing

Thank you for your interest in contributing to _Computational Statistics with R_!

The content of this repository will be included in a book to be published,
and hence we need to own the copyright for the complete content. The online
version will continue to be freely available and we will acknowledge all
contributions.

To confirm that you agree to surrender your copyright, please include the
phrase "I assign the copyright of this contribution to the book authors".

If you have any questions, feel free to raise them in the issue, pull
request, or contact the maintainers directly.

## How to contribute

If you find typos, errors, or have suggestions for improvements:

1. **Open an issue** on GitHub describing the problem or suggestion.
2. **Submit a pull request** with your proposed changes.

Please include the copyright assignment phrase above in your pull request.
